HTML:
  -> Hyper text markup language that means more than text that can be dispalyed on the web page.
  -> It is used for dispaly the content in the web page.
  -> HTML is the standard markup language for creating Web pages.
  -> HTML describes the structure of a Web page.
  -> HTML elements tell the browser how to display the content.
   

HTML Element:
 

-> HTML element is defined by a start tag, some content, and an end tag.


Tags:

 The <!DOCTYPE html> declaration defines that this document is an HTML5 document that means which version we are using in html we have multiple version but now we are using html5.
 The <html> element is the root element of an HTML page
 The <head> element contains meta information about the HTML page
 The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
 The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
 The <h1> element defines a large heading
 The <p> element defines a paragraph

Block Level Elements:

   -> Block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
   -> A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
   -> Two commonly used block elements are: <p> and <div>.
   -> It does not allow side  content always start with new line only.

Inline Elements: 

    -> Inline element does not start on a new line.
    -> Inline element only takes required width only.
    -> It allows side content also if you have required width.
